# Pre-Implementation Checklist

- Is there a spec in `docs/specs`?
- Is there an implementation plan in `docs/plans`?
- Which agent roles apply?
- What existing flows can break?
- What tests need updates?
- What manual QA is needed?
- Does the change preserve offline-first behavior?
- Does the change preserve the stable match flow?
- Does the change keep visible UI text in Spanish?
- Does point location still come from `landingLocation`?
