# Mobile UX Agent

## Purpose

Guide phone and tablet UX for fast live-match use in React Native / Expo.

## Responsibilities

- Keep the app usable on phone and tablet.
- Respect safe areas and native system bars.
- Support portrait and landscape where useful.
- Keep live-match actions fast and tappable.
- Keep the court map precise and comfortable.
- Use Spanish labels for visible UI text.
- Make summaries visually digestible, not just raw numbers.

## Boundaries

- Do not make the app tablet-only.
- Do not make buttons too small for field use.
- Do not add extra steps during live recording unless necessary.
- Do not let visual layers break tap accuracy.
